# 给 Web 开发者的 Python 速通指南

本项目是一份为有 JavaScript/TypeScript 背景的 Web 开发者量身定制的 Python 教程。

我们的目标是利用你已有的编程知识，通过类比和对比，快速掌握 Python，并将其应用于数据抓取和分析领域。
